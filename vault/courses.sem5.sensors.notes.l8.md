## LTSpice vs TinkerCAD

| LTSpice                      | TinkerCAD                             |
| ---------------------------- | ------------------------------------- |
| Good circuit schematics      | Components resemble actual components |
| Good for analog circuit sims | Good for digital circuit sims         |
|                              | Can do analog with limitations        |
